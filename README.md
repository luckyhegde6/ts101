# 🧮 TypeSafe Calculator (Week 1 - TypeScript Basics)

A simple calculator built with TypeScript to learn core TS concepts like types, interfaces, and compiler setup. Uses Jest for unit testing.

---

## Features

-   Type Safety
-   Easy to Use
-   Extensible
-   Unit tests with Jest

## 🚀 Getting Started

1️⃣ Clone the repository

2️⃣ Install Dependencies

```bash
npm install
```

3️⃣ Build the Project

```bash
npm run build
```

4️⃣ Run the Calculator

```bash
npm start
```

## Usage

-   🧩 Example Run

```bash
Enter operation (add, subtract, multiply, divide): add
Enter first number: 10
Enter second number: 5
Result: 15
```

## 🧪 Testing

To run the unit tests, use the following command:

```bash
npm test
```

This will execute the test suite using Jest.
