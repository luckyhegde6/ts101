# TypeScript Learning Notes

## What is TypeScript?
TypeScript is a superset of JavaScript that adds static type checking, making it easier to detect bugs at compile time.

## Key Concepts
- **Types**: Define the kind of data (number, string, boolean, etc.)
- **Interfaces**: Contracts for object structure.
- **Union Types**: Restrict a variable to multiple specific values.
- **Compiler**: `tsc` checks types and converts TS → JS.
- **tsconfig.json**: Configuration file for the TypeScript compiler.

## Why It Matters
TypeScript helps you write **self-documenting**, **scalable**, and **refactor-safe** code.
