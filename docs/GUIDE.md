# Developer Guide

### Run in Dev Mode
```bash
npx ts-node src/index.ts
```

### Rebuild on Save
Install ts-node-dev for live reloads.
```bash
npm install -D ts-node-dev
npm run dev
``` 

## Troubleshooting
Common Errors

TS2307: Cannot find module: Check import paths.

Type mismatch: Ensure variables match declared types.
